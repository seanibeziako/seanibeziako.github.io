Detection Rules / Mapped to MITRE ATT&CK
T1110.003 Building

Password Spray / Windows Logon Failures

Triggers when 10+ failed logons (EventID 4625) target 5+ unique accounts from a single source IP within a 5-minute window.

Source: Windows Security Log Rule: Sigma + Wazuh correlation
T1059.001 Building

Encoded PowerShell Execution

Detects PowerShell launched with -EncodedCommand, -enc, or base64 payloads via Sysmon Event 1, a common AV-evasion pattern in commodity malware.

Source: Sysmon Event 1 Rule: Sigma (proc_creation_win_powershell_enc)
T1003.001 Planned

LSASS Memory Access

Flags processes opening a handle to lsass.exe with suspicious GrantedAccess masks (0x1010, 0x1410) via Sysmon Event 10. Classic credential dumping signal (Mimikatz, procdump).

Source: Sysmon Event 10 Rule: Sigma (sysmon_lsass_memdump)
T1053.005 Planned

Suspicious Scheduled Task Creation

Alerts on schtasks.exe invocations creating tasks that run SYSTEM-level binaries or point to user-writable paths. A common persistence mechanism (EventID 4698).

Source: Windows Security Log Rule: Sigma (win_schtasks_persistence)
Sample Detection (Sigma YAML)
title: Potential Password Spray via Failed Logons
id: a9b1c3f0-lab-0001
status: experimental
description: Detects password spray attempts - many failed logons
  against multiple distinct accounts from a single source.
logsource:
  product: windows
  service: security
detection:
  selection:
    EventID: 4625
  timeframe: 5m
  condition: selection | count(TargetUserName) by IpAddress > 5
level: medium
tags:
  - attack.credential_access
  - attack.t1110.003
